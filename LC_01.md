#### Leetcode 01 [Two Sum](https://leetcode.com/problems/two-sum/description/)

Brute Force: O(n2) runtime, O(1) space
```java
//Time Complexity: O(N^2)
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int n = nums.length;

        for(int i = 0; i < n; i++){
            for(int j = i + 1; j < n; j++){
                if(nums[i] + nums[j] == target){
                    return new int[]{i, j};
                }
            }
        }
        return new int[0];
    }
}
```
